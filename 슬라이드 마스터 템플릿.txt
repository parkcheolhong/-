## 🧾 슬라이드 마스터 템플릿 구조

### 🔹 1. 표지 마스터

- **요소**: 브랜드 로고 / 발표 제목 / 발표자 이름 / 날짜  
- **배경**: 브랜드 블루 (#3A6EFF) 또는 감정별 그라데이션  
- **폰트**: Pretendard Bold / 28pt  
- **아이콘**: 🌍🎴🧠🎤  
- **레이아웃**: 중앙 정렬 / 하단에 발표자 정보

---

### 🔹 2. 제목 + 본문 마스터

- **용도**: 브랜드 철학 / 기능 설명 / 실행 전략  
- **레이아웃**: 좌측 제목 / 우측 본문 또는 카드형  
- **폰트**: 제목 24pt / 본문 16pt / 강조 18pt  
- **색상**: 밝은 배경 (#F9FAFB) + 감정별 강조 컬러  
- **구성**:
  ```
  ┌──────────────┐
  │ 제목         │
  ├──────────────┤
  │ 본문 내용     │
  │ - 핵심 요약   │
  │ - 흐름 설명   │
  └──────────────┘
  ```

---

### 🔹 3. 이미지 + 캡션 마스터

- **용도**: Figma 썸네일 / UI 시안 / 기능 데모  
- **레이아웃**: 이미지 좌측 / 캡션 우측 또는 하단  
- **구성**:
  ```
  ┌──────────────┐
  │ [이미지]     │
  ├──────────────┤
  │ 캡션 설명     │
  └──────────────┘
  ```

---

### 🔹 4. 타임라인 마스터

- **용도**: 피드백 반영 흐름 / 실행 일정  
- **레이아웃**: 수직 또는 수평 타임라인  
- **요소**: 날짜 / 기능명 / 상태 이모지 / 담당자  
- **예시**:
  ```
  [2025-09-28] ─ 표현 카드 컬러 개선 ─ ✅ 완료
  [2025-09-30] ─ 챌린지 XP 기준 검토 ─ 🛠️ 진행 중
  ```

---

### 🔹 5. 카드형 마스터

- **용도**: 기능 요약 / 확장 계획 / 팀별 피드백  
- **레이아웃**: 2~3단 카드 / 아이콘 + 제목 + 설명  
- **예시**:
  ```
  ┌────────────┐ ┌────────────┐
  │ 🎨 표현 카드 │ │ 🧠 챌린지 흐름 │
  │ 설명 요약   │ │ 설명 요약   │
  └────────────┘ └────────────┘
  ```

---

### 🔹 6. 마무리 메시지 마스터

- **용도**: 발표 종료 / 브랜드 메시지 강조  
- **배경**: 감정별 컬러 그라데이션  
- **폰트**: Pretendard Bold / 32pt  
- **메시지**:
  ```
  지금까지 만든 모든 것,  
  이제 세상과 연결될 시간입니다.
  ```
