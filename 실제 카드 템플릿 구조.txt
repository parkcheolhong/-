## 🧾 Notion 카드 템플릿 구조

```markdown

관리자 대시보드 모바일

### 🧭 기능 설명
사용자 현황, 콘텐츠 흐름, 피드 관리 기능을 모바일 환경에 최적화한 관리자용 UI 시안입니다.

### 🔗 Figma 링크
[Figma 프레임 바로가기](https://www.figma.com/file/abc123?node-id=10%3A24)

### 📷 썸네일 이미지
(이미지 삽입 또는 PNG 업로드)

### 🧑 담당자
HOUSING

### 📅 최종 수정일
2025-10-03

### ✅ 진행 상태
✅ 완료 / 🛠️ 진행 중 / 🧪 테스트 예정 / 🧠 기획 중

### 📝 피드백 메모
- 사용자 목록 테이블에서 역할 배지 색상 강조 필요  
- 콘텐츠 흐름 그래프는 Pie → Bar로 변경 제안  
- 모바일에서 버튼 간격 44px 이상 유지

---
```

> 📌 이 템플릿은 Notion 갤러리 뷰에서 카드 클릭 시 나타나는 상세 페이지로 설정됩니다.  
> 썸네일은 Figma에서 `Export Frame → PNG`로 추출 후 Notion에 직접 업로드하거나 드래그 삽입 가능해요.

## 🎯 확장 팁

- 템플릿 버튼으로 자동 생성 → 기능별 카드 빠르게 추가  
- `상태`, `담당자`, `기능 그룹` 속성으로 필터링 가능  
- `피드백 메모`는 팀원별 댓글 또는 체크리스트로 확장 가능  
- 발표/제안서 제작 시 카드 내용 그대로 복사 가능
