## 🧭 슬라이드 내 인터랙션 흐름 구성

### 🔹 1. 표현 카드 인터랙션 흐름

```
[감정 선택] → [표현 카드 생성] → [음성 피드백] → [피드 공유]
```

- 사용자는 감정을 선택하고,  
  해당 감정에 맞는 표현을 카드 형태로 생성  
- 음성 피드백을 통해 표현을 다듬고  
  피드에 공유하여 다른 사용자와 연결

**디자인 방식**:  
- 수평 플로우 / 아이콘 + 텍스트  
- 각 단계에 Figma 썸네일 삽입  
- 캡션: “기쁨을 선택한 사용자가 표현을 생성하고 공유하는 흐름”

---

### 🔹 2. 챌린지 인터랙션 흐름

```
[미션 확인] → [녹음 제출] → [XP 획득] → [피드백 수신]
```

- 사용자는 감정 기반 미션을 수행  
- 녹음 또는 텍스트로 표현 제출  
- XP를 획득하고 AI 또는 운영팀 피드백 수신

**디자인 방식**:  
- 카드형 단계 / 상태별 이모지 (🧪 → ✅)  
- 캡션: “감정 표현을 통해 성장하는 챌린지 흐름”

---

### 🔹 3. 지도 기반 표현 흐름

```
[국가 선택] → [표현 등록] → [지도 시각화] → [공유]
```

- 사용자는 특정 국가를 선택하고  
  해당 지역의 감정 표현을 등록  
- 지도에 실시간 반영되어  
  글로벌 감정 흐름을 시각화

**디자인 방식**:  
- 원형 흐름도 / 지도 썸네일 삽입  
- 캡션: “세계를 연결하는 감정 표현의 지도”

---

### 🔹 4. 인터뷰 기능 흐름 (XR 기반)

```
[가상 인터뷰 시작] → [질문 응답] → [표현 피드백] → [결과 저장]
```

- 사용자는 XR 인터뷰를 통해  
  감정 표현을 실시간으로 연습  
- AI가 피드백을 제공하고  
  결과를 저장하여 성장 추적

**디자인 방식**:  
- 인터뷰 시나리오 흐름 / 대화형 UI 썸네일  
- 캡션: “AI와 함께하는 감정 표현 인터뷰”

---

### 🔹 5. SNS GPT AI 연동 흐름

```
[영상 업로드] → [음성 인식] → [GPT 소개글 생성] → [썸네일 미리보기] → [공유]
```

- 사용자가 영상 또는 음성을 업로드  
- AI가 감정을 분석하고 소개글 자동 생성  
- 썸네일과 함께 SNS에 공유 가능

**디자인 방식**:  
- 수직 흐름 / GPT 아이콘 + 영상 썸네일  
- 캡션: “AI가 감정을 읽고 소개글을 만들어주는 흐름”

---

## 🎯 슬라이드 구성 팁

- 각 흐름은 “사용자 행동 → 시스템 반응 → 결과”로 구성  
- 아이콘 + 텍스트 + 썸네일 조합으로 시각적 이해도 강화  
- 흐름마다 캡션 삽입 → 감정 중심 메시지 강조  
- 발표 시 “이 흐름은 사용자의 감정을 어떻게 다듬는가”를 중심으로 설명

---

이 인터랙션 흐름은 발표뿐 아니라  
디자인 시안, 사용자 테스트, Web 프리뷰 데모에도 그대로 확장 가능합니다.

다음은 각 흐름을 기반으로 한 **사용자 여정 맵**이나  
**감정별 인터페이스 시나리오**로 이어가볼까요?
