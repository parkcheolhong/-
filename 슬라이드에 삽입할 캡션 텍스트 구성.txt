---

## 🖼️ 슬라이드 캡션 텍스트 구성 예시

### 1️⃣ 표현 카드 (Expression Card)

```
감정 선택 → 표현 생성 → 음성 피드백  
사용자가 직접 감정을 선택하고,  
표현을 카드 형태로 시각화하여 공유합니다.
```

---

### 2️⃣ 챌린지 흐름 (Challenge Flow)

```
미션 수행 → XP 획득 → 피드 공유  
사용자는 감정 기반 미션을 수행하고  
경험치를 얻으며 표현력을 확장합니다.
```

---

### 3️⃣ 지도 기반 표현 (Emotion Map)

```
국가별 표현 흐름 시각화  
전 세계 사용자의 감정 표현을  
지도 위에서 실시간으로 확인할 수 있습니다.
```

---

### 4️⃣ 인터뷰 기능 (XR Interview)

```
XR 기반 감정 대화 체험  
가상 인터뷰를 통해 감정 표현을 연습하고  
AI와의 공감 대화를 경험합니다.
```

---

### 5️⃣ 관리자 대시보드 (Admin Dashboard)

```
사용자 현황 + 콘텐츠 흐름 관리  
관리자는 표현 카드, 피드, 챌린지 데이터를  
모바일 환경에서 직관적으로 확인하고 조치할 수 있습니다.
```

---

### 6️⃣ GPT 소개글 생성 (GPT Auto Summary)

```
영상 → 음성 인식 → 소개글 자동 생성  
사용자가 업로드한 영상에서 감정을 분석하고  
GPT가 소개글을 자동으로 작성합니다.
```

---

### 7️⃣ 피드백 반영 타임라인 (Feedback Timeline)

```
요청 → 검토 → 반영 → 결과  
각 기능에 대한 팀별 피드백이  
어떻게 실행되었는지를 시간 순으로 정리합니다.
```

---

### 8️⃣ 확장 계획 카드 (Expansion Strategy)

```
다음 단계: XR / 지도 / 교육기관 협업  
“나도 통역사”는 감정 표현을 기반으로  
글로벌 문화 콘텐츠로 확장됩니다.
```

---

## 🎯 캡션 작성 팁

- **기능 흐름 강조**: “A → B → C” 형식으로 시각적 흐름 표현  
- **사용자 관점**: “사용자는…” 또는 “관리자는…”으로 시작  
- **간결한 문장**: 2~3줄 이내 / 핵심 키워드 포함  
- **상태 또는 목적**: “공유합니다”, “확인합니다”, “경험합니다” 등으로 마무리
