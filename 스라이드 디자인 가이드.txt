## 🎨 슬라이드 디자인 시안 구성

### ✅ 전체 스타일 가이드

- **비율**: 16:9 와이드 화면  
- **폰트**: Noto Sans / Pretendard / 고딕체 계열 (명료하고 감성적)  
- **컬러 팔레트**:  
  - 감정 카드 기반 색상 (예: 따뜻한 오렌지, 신뢰의 블루, 창의적 퍼플)  
  - 배경은 밝은 톤, 텍스트는 대비되는 진한 색상  
- **아이콘 스타일**: 선형 아이콘 + 감정 카드 이미지  
- **비주얼 요소**: 지도 흐름도, 인터뷰 인용, GPT 아이콘, XR 체험 이미지

---

## 🧩 슬라이드별 디자인 시안

### 1️⃣ 브랜드 개요
- 배경: 감정 카드 이미지 모자이크  
- 중앙 텍스트: “나도 통역사” 로고 + 슬로건  
- 하단: 플랫폼 철학 요약 (표현권, 연결, 성장)

---

### 2️⃣ 콘텐츠 구조
- 4개 콘텐츠 블록 (카드 / 인터뷰 / 피드 / 리포트)  
- 각 블록에 아이콘 + 짧은 설명  
- 흐름선으로 연결 (→)

---

### 3️⃣ 실행 흐름
- 4단계 타임라인 (디자인 → 배포 → 테스트 → AI 연동)  
- 각 단계에 시안 이미지 또는 아이콘  
- 배경: 흐름도 스타일

---

### 4️⃣ GitHub 구조
- 디렉토리 트리 시각화  
- 각 폴더에 기능 설명 아이콘  
- 배경: 코드 스타일 텍스처

---

### 5️⃣ 사용자 경험
- 모바일 화면 시안 3개 (카드 선택 / 챌린지 / 인터뷰 XR)  
- 각 화면에 설명 캡션  
- 배경: 사용자 손 이미지 또는 XR 체험 장면

---

### 6️⃣ AI 기능
- GPT 아이콘 + 음성 인식 마이크 + 영상 썸네일  
- 기능 흐름도: 영상 → 음성 인식 → 소개글 생성 → 미리보기  
- 배경: AI 회로 스타일 그래픽

---

### 7️⃣ 콘텐츠 패키지
- PDF/PPT 썸네일 이미지  
- 캠페인 키트 구성 요소 아이콘 (카드 / 인용 / 영상 템플릿)  
- 배경: 문서 스타일 패턴

---

### 8️⃣ 협업 방식
- GitHub + Notion 로고  
- Web 프리뷰 데모 화면 캡처  
- 협업 흐름도: 개발자 ↔ 콘텐츠 팀 ↔ 전략팀

---

### 9️⃣ 기대 효과
- 3개 원형 블록: 표현권 확산 / 글로벌 연결 / 창작자 성장  
- 각 블록에 인물 이미지 또는 지도 시각화  
- 배경: 따뜻한 연결선 스타일

---

### 🔟 실현 → 확산 → 연결
- 3단계 피라미드 또는 화살표 흐름  
- 각 단계에 핵심 키워드 + 아이콘  
- 마무리 메시지 강조: “지금은 세상과 연결될 시간입니다”
