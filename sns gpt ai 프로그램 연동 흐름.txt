## 🤖 SNS GPT AI 프로그램 연동 흐름

### 1️⃣ 입력 방식

- **영상 업로드**: 사용자가 촬영한 영상 파일 선택  
- **직접 촬영**: Web 프리뷰에서 카메라 접근 → 실시간 녹화  
- **음성 입력**: 마이크로 감정 표현 녹음

---

### 2️⃣ 처리 모드

- **자동 모드**:  
  - 영상 분석 → 감정 인식 → GPT 소개글 자동 생성  
  - 썸네일 자동 추출 + 오디오 분리  
  - 음성 명령 인식 (예: “슬픔 표현 영상 보여줘”)

- **수동 모드**:  
  - 사용자가 직접 키워드 입력  
  - GPT가 제안하는 소개글 중 선택  
  - 썸네일/오디오 수동 편집 가능

---

### 3️⃣ 출력 결과

- **소개글**: GPT 기반 감정 중심 설명문  
- **썸네일**: 영상에서 감정이 잘 드러나는 장면 자동 추출  
- **오디오**: Whisper 기반 음성 인식 + 재생 기능  
- **명령 인식**: 음성으로 영상 탐색 및 기능 호출

---

### 4️⃣ 오류 대응 및 개선

- 영상/음성 인식 실패 시 자동 안내  
- GPT 결과가 부적절할 경우 수정 제안  
- 기능 개선 피드백 수집 → 사용자 리포트 반영

---

## 🔗 연동 방식 제안

- **GPT API**: OpenAI 또는 Azure 기반  
- **Whisper API**: 음성 인식 및 텍스트 변환  
- **FFmpeg**: 영상 썸네일 및 오디오 추출  
- **React + Node.js**: Web 프리뷰와 연동
