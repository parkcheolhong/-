# 나도 통역사: 글로벌 표현력 성장 플랫폼

## 프로젝트 개요
“나도 통역사”는 감정과 표현을 중심으로 세계를 연결하는 문화 플랫폼입니다.  
앱과 콘텐츠를 통해 표현권을 확산하고, 인터뷰 기반 XR 체험과 AI 연동 콘텐츠를 제공합니다.

## 디렉토리 구조

- `Frontend/`: 감정 카드 UI, 인터뷰 화면, 피드, 리포트
- `backend/`: 사용자 DB, 표현 저장, XP 시스템, 배지 인증
- `ai/`: GPT 질문 생성, 음성 인식, 소개글 자동 생성
- `assets/`: 카드 이미지, 영상 샘플, 인터뷰 인용, 캠페인 키트
- `docs/`: 실행 흐름, 콘텐츠 맵, README, 제안서, 일정표

## 실행 흐름
1. 디자인 시안 제작
2. 배포 채널 구성
3. 사용자 테스트 준비
4. SNS GPT AI 프로그램 연동
5. 교육기관 콘텐츠 패키지 구성

## 기술 스택 제안
- Frontend: React / Flutter
- Backend: Node.js / Firebase
- AI: OpenAI GPT / Whisper / TTS
- Web 프리뷰: Next.js / Vercel

## 협업 방식
- GitHub 저장소 기반 브랜치 관리
- Notion 전략 페이지 연동
- PDF/PPT 콘텐츠 패키지 배포
