## 🧱 실제 컴포넌트 프레임 예시: `AdminDashboard_Mobile`

### 🔹 프레임 크기

- 이름: `AdminDashboard_Mobile`  
- 크기: `360 x 640` (iPhone SE 기준)  
- Layout Grid: 4-column / 8px gutter / 16px margin  
- Auto Layout: Vertical / Padding 16px / Spacing 12px

---

### 🔹 컴포넌트 배치

```
┌────────────────────────────┐
│ Header                     │
│ [☰] 관리자 대시보드         │
├────────────────────────────┤
│ StatCardGroup              │
│ ┌ StatCard ┐ ┌ StatCard ┐ │
│ └──────────┘ └──────────┘ │
├────────────────────────────┤
│ SectionTitle: 콘텐츠 흐름   │
│ ChartBlock (PieChart)      │
│ ChartBlock (BarChart)      │
├────────────────────────────┤
│ SectionTitle: 사용자 목록   │
│ UserRowList                │
│ ┌ UserRow ┐                │
│ │ 아바타 닉네임 역할 상태 │
│ │ [수정] [비활성화]       │
│ └────────┘                │
├────────────────────────────┤
│ TabBar                     │
│ [대시보드] [피드관리] [설정]│
└────────────────────────────┘
```

---

### 🔹 주요 컴포넌트 속성

#### ✅ StatCard

- Variants: `type = 전체 / 활성 / 비활성 / Creator / Viewer`  
- Props: `icon`, `label`, `value`, `colorTheme`  
- Auto Layout: Horizontal / Center / Padding 12px

#### 📊 ChartBlock

- Variants: `type = pie / bar / line`  
- Props: `title`, `data`, `colorSet`  
- Auto Layout: Vertical / Padding 16px

#### 👤 UserRow

- Props: `avatar`, `nickname`, `role`, `status`, `actions[]`  
- Actions: `Edit`, `Deactivate`, `Reactivate`  
- Auto Layout: Horizontal / Space Between

#### 🧭 TabBar

- Variants: `activeTab = dashboard / feed / settings`  
- Props: `icon`, `label`, `isActive`  
- Auto Layout: Horizontal / Fill Container

## 🧠 Figma 팁

- 모든 컴포넌트는 `Variants`로 상태 관리  
- `Auto Layout` + `Constraints`로 반응형 대응  
- `Component Set`으로 그룹화 → 라이브러리 공유  
- `Prototype` 탭에서 인터랙션 흐름 연결 가능
