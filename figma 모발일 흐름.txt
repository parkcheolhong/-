## 🎨 Figma 모바일 시안 흐름

### 🧩 1. 프레임 구성

| 프레임 이름 | 설명 |
|-------------|------|
| `Login` | 관리자 로그인 화면  
| `Dashboard` | 사용자 현황 + 콘텐츠 흐름 카드  
| `UserList` | 사용자 목록 테이블형 리스트  
| `UserDetail` | 사용자 상세 정보 + 역할 수정  
| `FeedModeration` | 표현 피드 관리 화면  
| `ChallengeLog` | 챌린지 수행 내역 + XP 기록  
| `MenuDrawer` | 사이드 메뉴 (통계 / 설정 / 로그아웃)

> 각 프레임은 360x640 기준으로 구성 (모바일 기본 해상도)

---

### 🧱 2. 컴포넌트 구조

| 컴포넌트 | 설명 |
|----------|------|
| `StatCard` | 사용자 수, 역할별 분포 등 요약 카드  
| `UserRow` | 사용자 목록 테이블의 한 줄  
| `ActionButton` | 수정 / 비활성화 / 재활성화 버튼  
| `FeedCard` | 표현 카드 + 좋아요/댓글 수  
| `ChartBlock` | 표현 흐름 그래프 (Chart.js 스타일)  
| `ModalConfirm` | 사용자 조치 시 확인 팝업

> 모든 컴포넌트는 Auto Layout 적용 → 반응형 대응

---

### 🔄 3. 인터랙션 흐름

```plaintext
Login → Dashboard → UserList → UserDetail → RoleEdit → ConfirmModal
                         ↘︎
                    FeedModeration → FeedCard → DeleteConfirm
                         ↘︎
                    ChallengeLog → XPChart → UserMissionDetail
```

> 주요 흐름은 버튼 클릭 → 프레임 전환 → 모달 팝업 방식  
> 드로어 메뉴는 `MenuDrawer`에서 슬라이드 인/아웃

---

### 🎨 4. 스타일 가이드

| 요소 | 값 |
|------|-----|
| 폰트 | Pretendard / Noto Sans KR  
| 색상 | 브랜드 블루 `#3A6EFF`, 감정별 테마 컬러  
| 버튼 | 기본 `44px` 높이 / 라운드형 / 그림자 있음  
| 아이콘 | Feather Icons / Emotion Emoji Set  
| 그래프 | 선형 / 원형 / 막대형 혼합

---

## 🧠 디자인 팁

- 감정 중심 플랫폼이므로 **색상과 아이콘으로 감정 표현 강조**  
- 관리자 UX는 **즉시 조치 가능 + 정보 요약 중심**  
- 모바일에서도 **터치 최적화 + 스와이프 인터랙션** 적용  
- Figma에서는 `Variants`로 컴포넌트 상태 관리 (활성/비활성/에러 등)
