## 🧩 팀별 피드백 공간 구성

### 🔹 1. 페이지 구조 제안

```
📁 “나도 통역사” 실행 전략
├── 🎨 디자인 시안 갤러리
├── 🧭 실행 흐름 요약
├── 🗂️ 기능별 시안 링크
├── 💬 팀별 피드백 공간
│   ├── 디자인팀 피드백
│   ├── 기획팀 피드백
│   ├── 개발팀 피드백
│   └── 운영팀 피드백
```

> 각 팀별 피드백 공간은 별도 섹션 또는 하위 페이지로 구성  
> Notion에서는 `Toggle`, `Table`, `Comment`, `Callout` 블록을 활용

---

### 🔹 2. 역할별 피드백 섹션 예시

#### 🎨 디자인팀 피드백

```markdown
🖼️ 표현 카드 UI
- 카드 간 간격 12px → 16px로 확대 제안
- 감정별 컬러 대비 부족 → 슬픔/분노 계열 강조 필요

📱 모바일 대시보드
- 버튼 터치 영역 44px 이상 유지
- PieChart → BarChart로 변경 시안 요청
```

#### 🧠 기획팀 피드백

```markdown
📌 챌린지 흐름
- XP 획득 기준 명확화 필요
- 실패 시 피드백 문구 → 감정 중심으로 재작성 제안

📌 인터뷰 기능
- GPT 소개글 → 감정별 톤 조절 옵션 추가 검토
```

#### 🧑‍💻 개발팀 피드백

```markdown
🔧 API 연동
- `/challenge/submit` → 음성 파일 처리 속도 개선 필요
- `/map/add` → 국가 코드 누락 시 오류 처리 추가

🔐 인증 흐름
- Refresh Token 만료 시 자동 로그아웃 처리 검토
```

#### 🧑‍🏫 운영팀 피드백

```markdown
📊 사용자 테스트
- 표현 카드 선택률 높은 감정: 기쁨, 외로움 → 콘텐츠 우선 배치 제안
- 피드 공유 기능 → SNS 연동 시 해시태그 자동 생성 검토

📣 캠페인 운영
- 지도 표현 → 국가별 감정 테마 컬러 적용 요청
```

### 🔹 3. 피드백 템플릿 버튼 예시 (Notion)

```markdown
📥 피드백 카드 템플릿

- 기능명:
- 시안 링크:
- 담당자:
- 피드백 내용:
- 우선순위: 🔴 높음 / 🟡 중간 / 🟢 낮음
- 상태: 🛠️ 진행 중 / ✅ 반영 완료 / 🧪 테스트 중
```

> 템플릿 버튼으로 각 팀이 빠르게 피드백 카드 생성 가능  
> 상태별 필터링 → 반영 여부 추적 가능

---

### 🔹 4. 실행 연결 방식

- 각 피드백은 해당 기능 시안 카드에 `@멘션` 또는 `링크 삽입`  
- 반영 완료 시 `✅ 상태 변경` + `담당자 코멘트`  
- 주간 회의 시 `피드백 요약 테이블`로 정리 가능
