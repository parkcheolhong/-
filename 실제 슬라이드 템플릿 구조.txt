## 🧾 슬라이드 템플릿 구조 (총 10장 기준)

| 슬라이드 번호 | 제목 | 구성 요소 |
|---------------|------|------------|
| 1 | 표지 | 브랜드 로고 / 발표 제목 / 발표자 이름 / 날짜  
| 2 | 브랜드 철학 | “나도 통역사”의 비전 / 키워드 / 감정 중심 플랫폼 설명  
| 3 | 전체 콘텐츠 흐름 | 표현 카드 / 챌린지 / 지도 / 인터뷰 / 피드 / 흐름도  
| 4 | UI 시안 요약 | Figma 썸네일 / 모바일/웹 시안 / 감정별 컬러 테마  
| 5 | 실행 전략 | GitHub / Notion / Web 프리뷰 / 배포 구조  
| 6 | 사용자 테스트 흐름 | 테스트 시나리오 / 감정 선택 → 표현 생성 → 공유  
| 7 | 피드백 반영 타임라인 | 요청 → 반영 → 결과 / 상태별 이모지 / 날짜 흐름  
| 8 | SNS GPT AI 연동 | 영상 업로드 → 음성 인식 → GPT 소개글 생성 흐름  
| 9 | 향후 확장 계획 | XR 인터뷰 / 글로벌 감정 지도 / 교육기관 협업  
| 10 | 마무리 메시지 | “세상을 연결하는 표현의 혁신” / 브랜드 슬로건 / 연락처

## 🎨 디자인 요소 제안

- 폰트: Pretendard / Noto Sans KR  
- 색상: 브랜드 블루 `#3A6EFF` + 감정별 테마 컬러  
- 아이콘: Feather Icons / Emotion Emoji Set  
- 레이아웃: 좌측 텍스트 + 우측 이미지 또는 카드형  
- 배경: 밝은 톤 (`#F9FAFB`) / 강조 슬라이드는 컬러 배경  
- 전환 효과: 페이드 / 슬라이드 / 줌 인 (감정 강조 시)

## 📌 슬라이드 제작 팁

- 각 슬라이드에 Figma 썸네일 또는 Notion 링크 삽입  
- 상태별 이모지 (✅ 🛠️ 🔴 🧪)로 실행력 강조  
- 발표 흐름은 “철학 → 구조 → 실행 → 피드백 → 확장”  
- 마지막 슬라이드는 감정 중심 메시지로 마무리
