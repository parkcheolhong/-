## 🧩 Web 프리뷰 인터랙션 예시

### 🎴 표현 카드

| 인터랙션 | 설명 |
|-----------|------|
| 카드 클릭 | 감정 선택 시 카드가 회전하며 표현 예시 출력 |
| 표현 예시 선택 | 텍스트/음성 입력창 활성화 |
| 피드백 버튼 | “이 표현이 어땠나요?” → 좋아요/별점/코멘트 입력 가능 |

---

### 🧠 챌린지

| 인터랙션 | 설명 |
|-----------|------|
| 미션 카드 등장 | 오늘의 표현 미션 자동 생성 (예: “기쁨을 표현해보세요”) |
| 음성 녹음 | 마이크 버튼 클릭 → 녹음 시작/종료 → XP 획득 |
| XP 게이지 | 미션 성공 시 게이지 애니메이션으로 상승

---

### 🌍 지도

| 인터랙션 | 설명 |
|-----------|------|
| 국가 클릭 | 해당 국가의 표현 카드 리스트 출력 |
| 사용자 표현 추가 | 텍스트/음성 입력 → 지도에 핀 추가됨 |
| 표현 흐름 보기 | 국가별 표현 변화 시각화 (시간순 애니메이션)

---

### 🎥 인터뷰

| 인터랙션 | 설명 |
|-----------|------|
| 질문 카드 클릭 | XR 인터뷰 영상 재생 시작 |
| 답변 입력 | 텍스트/음성으로 감정 표현 → GPT 소개글 생성 |
| 소개글 미리보기 | 생성된 소개글을 카드 형태로 확인 가능

---

### 📰 피드

| 인터랙션 | 설명 |
|-----------|------|
| 표현 카드 리스트 | 사용자들이 공유한 표현 카드 스크롤 가능 |
| 좋아요/댓글 | 아이콘 클릭 → 반응 입력 → 실시간 반영 |
| 카드 클릭 | 상세 표현 + 사용자 코멘트 보기

---

### 🎙️ SNS GPT AI 기능

| 인터랙션 | 설명 |
|-----------|------|
| 영상 업로드 | 파일 선택 또는 직접 촬영 → 썸네일 자동 추출 |
| 음성 입력 | 마이크 녹음 → Whisper로 텍스트 변환 |
| 소개글 생성 | GPT가 감정 중심 소개글 자동 생성 → 수정 가능
| 명령 인식 | “다음 영상 보여줘” → 음성 명령으로 탐색 가능

---

## 🎯 기술 구현 팁

- JavaScript 이벤트: `onclick`, `oninput`, `onrecord`, `onhover`  
- 애니메이션: `CSS transition`, `GSAP`, `Framer Motion`  
- 지도: `Leaflet.js`, `D3.js`  
- 음성 인식: `Whisper API`, `Web Speech API`  
- GPT 연동: `OpenAI API`, `Azure OpenAI`
