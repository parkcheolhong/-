## 🧾 실제 PPT 제작 흐름

### 1️⃣ 슬라이드 구조 설계

- 총 10장 기준으로 구성  
- 흐름: 브랜드 철학 → 기능 흐름 → 디자인 시안 → 실행 전략 → 사용자 테스트 → 피드백 반영 → 확장 계획 → 마무리 메시지  
- 슬라이드 제목 및 순서 확정  
- 각 슬라이드에 들어갈 핵심 메시지 요약

---

### 2️⃣ 마스터 템플릿 적용

- 폰트: Pretendard / Noto Sans KR  
- 색상: 브랜드 블루 (#3A6EFF) + 감정별 테마 컬러  
- 배경: 밝은 톤 (#F9FAFB) 또는 감정 그라데이션  
- 아이콘: 기능별 이모지 + 상태 이모지 (✅ 🛠️ 🔴 🧪)  
- 레이아웃: 카드형 / 타임라인 / 그래프 / 이미지 + 캡션

---

### 3️⃣ 콘텐츠 삽입

- 텍스트: 슬라이드별 핵심 메시지 삽입  
- 이미지: Figma 썸네일 PNG 삽입 (1280x720 권장)  
- 캡션: 기능 흐름 / 사용자 반응 / 개선 제안 요약  
- 그래프: 반영 현황 요약 (원형 또는 막대 그래프)  
- 타임라인: 피드백 반영 흐름 (날짜 + 상태)

---

### 4️⃣ 발표용 최적화

- 전환 효과: 페이드 인 / 카드 슬라이드 / 강조 줌 인  
- 발표자 노트: 각 슬라이드별 설명 멘트 작성  
- 테스트: 슬라이드 흐름 점검 / 이미지 해상도 확인  
- 배포: PDF로 저장 / 발표용 노트 포함 / IR 제안서 연동

---

## 🎯 제작 팁

- 슬라이드 제목은 감정 중심 키워드로 구성 (예: “표현을 통해 연결하다”)  
- 기능별 색상 태그로 시각적 흐름 강조  
- 발표자 멘트는 사용자 여정 기반으로 구성  
- 발표 마무리는 “세상과 연결될 시간입니다” 메시지로 감정적 여운 남기기
