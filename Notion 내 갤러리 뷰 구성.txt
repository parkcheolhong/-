## 🖼️ Notion 갤러리 뷰 구성 예시

### 🔹 갤러리 뷰 설정

- 데이터베이스 이름: `🎴 디자인 시안 갤러리`  
- 뷰 타입: `Gallery`  
- 카드 크기: `Medium` 또는 `Large`  
- 속성 표시: `제목`, `설명`, `링크`, `상태`, `담당자`

### 🔹 필터/정렬 기능

- 필터: `상태 = 진행 중`, `담당자 = HOUSING`  
- 정렬: `제목 오름차순`, `최근 수정일 기준`  
- 그룹화: `기능별` 또는 `플랫폼별` (모바일 / 웹 / XR)

---

### 🔹 활용 방식

- 전략 페이지 상단에 `갤러리 뷰` 삽입 → 전체 흐름 시각화  
- 각 카드 클릭 시 → 상세 설명 + Figma 링크 + 피드백 공간  
- 팀원별 담당자 지정 → 진행 상황 추적  
- 발표/제안서 제작 시 → 썸네일 그대로 활용 가능

## 🎯 디자인 협업 팁

- 갤러리 뷰는 디자인 시스템의 시각적 맵 역할  
- 기능별 흐름을 한눈에 파악 → 기획/개발/디자인 연결  
- Notion에서 `템플릿 버튼`으로 새 카드 생성 자동화 가능  
- Figma 링크는 `node-id` 기준으로 정확하게 연결
