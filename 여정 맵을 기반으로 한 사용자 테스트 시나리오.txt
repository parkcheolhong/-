각 기능에 대해 사용자가 어떤 감정 상태에서 출발해 어떤 행동을 하고, 어떤 결과를 경험하는지를

---

## 🧪 사용자 테스트 시나리오: 기능별 구성

---

### 1️⃣ 표현 카드

- **목표**: 사용자가 감정을 선택하고 표현 카드를 생성 및 공유할 수 있는지 확인  
- **시나리오 흐름**:  
  1. 사용자는 “외로움”을 선택한다  
  2. 관련 표현 카드가 제시된다  
  3. 음성 피드백을 듣고 표현을 수정한다  
  4. 피드에 공유한다  
- **테스트 항목**:  
  - 감정 선택 UI 반응  
  - 카드 생성 속도 및 정확도  
  - 음성 피드백 음질 및 내용  
  - 공유 후 피드 반영 여부  
- **기대 결과**:  
  - 사용자가 표현을 직관적으로 생성하고  
  - 피드백을 반영해 공유까지 완료

---

### 2️⃣ 챌린지

- **목표**: 사용자가 미션을 수행하고 XP를 획득하며 표현력을 확장할 수 있는지 확인  
- **시나리오 흐름**:  
  1. 사용자는 “기쁨을 표현하라”는 미션을 확인한다  
  2. 녹음 또는 텍스트로 응답한다  
  3. XP를 획득하고 피드백을 받는다  
  4. 결과를 피드에 공유한다  
- **테스트 항목**:  
  - 미션 UI 가독성  
  - 녹음 기능 안정성  
  - XP 계산 로직  
  - 피드백 정확도 및 표현력 추적  
- **기대 결과**:  
  - 사용자가 감정 표현을 통해 성취감을 느끼고  
  - 시스템이 이를 정확히 반영

---

### 3️⃣ 지도 기반 표현

- **목표**: 사용자가 국가별 표현을 등록하고 지도에 반영되는지 확인  
- **시나리오 흐름**:  
  1. 사용자는 “브라질”을 선택한다  
  2. 해당 지역 표현을 등록한다  
  3. 지도에 실시간 반영된다  
  4. 다른 사용자와 공유한다  
- **테스트 항목**:  
  - 국가 선택 UI 반응  
  - 표현 등록 기능  
  - 지도 시각화 정확도  
  - 공유 기능 안정성  
- **기대 결과**:  
  - 사용자가 글로벌 감정 흐름에 기여하고  
  - 지도에서 이를 직관적으로 확인

---

### 4️⃣ 인터뷰 기능 (XR 기반)

- **목표**: 사용자가 가상 인터뷰를 통해 감정 표현을 연습하고 피드백을 받을 수 있는지 확인  
- **시나리오 흐름**:  
  1. 사용자는 인터뷰를 시작한다  
  2. 질문에 음성으로 응답한다  
  3. AI가 피드백을 제공한다  
  4. 결과를 저장하고 다음 인터뷰를 추천받는다  
- **테스트 항목**:  
  - 인터뷰 UI 흐름  
  - 음성 인식 정확도  
  - 피드백 내용의 감정 적합성  
  - 결과 저장 및 추천 기능  
- **기대 결과**:  
  - 사용자가 감정 표현을 반복적으로 연습하며  
  - 성장 흐름을 체감할 수 있음

---

### 5️⃣ SNS GPT AI 연동

- **목표**: 사용자가 영상/음성을 업로드하고 GPT가 소개글을 생성하는 흐름을 검증  
- **시나리오 흐름**:  
  1. 사용자는 영상 파일을 업로드한다  
  2. 음성 인식이 시작된다  
  3. GPT가 소개글을 자동 생성한다  
  4. 썸네일과 함께 SNS에 공유한다  
- **테스트 항목**:  
  - 영상 업로드 안정성  
  - 음성 인식 정확도  
  - GPT 소개글 품질  
  - 썸네일 미리보기 및 공유 기능  
- **기대 결과**:  
  - 사용자가 감정을 담은 콘텐츠를  
  - AI의 도움으로 쉽게 공유

---

## 🎯 테스트 시나리오 활용 팁

- 각 시나리오는 사용자 여정 맵과 연결  
- 테스트 결과는 피드백 반영 타임라인에 기록  
- Notion 또는 Google Form으로 테스트 결과 수집  
- 사용자 인터뷰와 병행 시 감정 중심 피드백 확보 가능
