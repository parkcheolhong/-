## 🧾 피드백 요약 슬라이드 (예시 텍스트 구성)

### 🎯 슬라이드 제목  
**사용자 테스트 결과 요약 & 피드백 반영 현황**

---

### 🧩 카드형 구성 예시 (슬라이드 내 2~3단 배치)

#### 🟦 표현 카드  
- **테스트 흐름**: 외로움 선택 → 카드 생성 → 음성 피드백 → 공유  
- **사용자 반응**: 표현 직관적 / 음성 피드백 유익 / 공유 쉬움  
- **발견된 문제**: 카드 간 간격 좁음 / 음성 음량 낮음  
- **개선 제안**: 카드 간격 확대 / 음량 조절 기능 추가  
- **반영 상태**: 🛠️ 진행 중 (디자인팀)

---

#### 🟨 챌린지  
- **테스트 흐름**: 기쁨 표현 미션 → 녹음 제출 → XP 획득 → 피드 공유  
- **사용자 반응**: 성취감 있음 / XP 시스템 이해 쉬움  
- **발견된 문제**: 녹음 중단 시 오류 발생  
- **개선 제안**: 녹음 재시작 기능 추가  
- **반영 상태**: ✅ 완료 (개발팀)

---

#### 🟩 지도 기반 표현  
- **테스트 흐름**: 브라질 선택 → 표현 등록 → 지도 반영 → 공유  
- **사용자 반응**: 시각화 흥미로움 / 국가별 흐름 이해 쉬움  
- **발견된 문제**: 국가 선택 시 로딩 지연  
- **개선 제안**: 캐시 처리 및 로딩 애니메이션 추가  
- **반영 상태**: 🧪 테스트 중 (개발팀)

---

#### 🟥 인터뷰 기능  
- **테스트 흐름**: 인터뷰 시작 → 질문 응답 → 피드백 수신 → 결과 저장  
- **사용자 반응**: AI 피드백 공감됨 / 반복 학습 유익  
- **발견된 문제**: 음성 인식 누락 발생  
- **개선 제안**: 음성 인식 정확도 향상 / 재녹음 기능 추가  
- **반영 상태**: 🔴 미반영 (기획팀)
