## 🧪 SNS GPT AI 기능 테스트 시뮬레이션

### 1️⃣ 입력 단계: 사용자 콘텐츠 제공

- **옵션 A**: 영상 파일 업로드 (예: “슬픔을 표현한 30초 영상”)  
- **옵션 B**: 직접 촬영 (Web 프리뷰에서 카메라 접근)  
- **옵션 C**: 음성 녹음 (마이크로 감정 표현)

> 예시: 사용자가 “나는 오늘 너무 외로웠어요”라고 말하며 녹음

---

### 2️⃣ 처리 단계: AI 분석 및 콘텐츠 생성

- **자동 모드**  
  - Whisper: 음성 → 텍스트 변환  
  - GPT: 감정 분석 → 소개글 자동 생성  
  - FFmpeg: 영상 썸네일 추출 + 오디오 분리

- **수동 모드**  
  - 사용자가 키워드 입력 (“외로움”, “밤”, “혼자”)  
  - GPT가 3가지 소개글 제안 → 선택

> 예시 GPT 출력:  
> “이 영상은 외로움이라는 감정을 담고 있습니다.  
> 밤의 고요함 속에서 혼자라는 감정이 어떻게 표현되는지를 보여줍니다.”

---

### 3️⃣ 출력 단계: 콘텐츠 결과 확인

- **소개글**: 감정 중심 설명문 자동 생성  
- **썸네일**: 영상에서 감정이 잘 드러나는 장면 자동 추출  
- **오디오**: 음성 인식 결과 + 재생 기능  
- **명령 인식**: “슬픔 영상 보여줘” → 해당 콘텐츠 탐색

---

### 4️⃣ 피드백 단계: 사용자 반응 및 개선

- 소개글 수정 제안 가능  
- 음성 인식 오류 시 자동 안내  
- 기능 개선 요청 수집 → 사용자 리포트 반영

---

## 🎯 테스트 시나리오 요약

| 단계 | 사용자 행동 | 시스템 반응 |
|------|--------------|--------------|
| 입력 | 영상/음성 제공 | 파일 수신 및 분석 시작 |
| 처리 | 자동/수동 선택 | GPT + Whisper + FFmpeg 처리 |
| 출력 | 결과 확인 | 소개글, 썸네일, 오디오 제공 |
| 피드백 | 수정/제안 | 시스템 개선 및 반영 |
