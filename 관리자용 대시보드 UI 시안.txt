## 🎯 기술 구현 팁

- 프론트엔드: React + Tailwind / Material UI  
- 백엔드: Express + JWT 인증 + Role 기반 접근  
- 그래프: Chart.js / Recharts  
- 테이블: DataGrid + 필터/검색 기능  
- 관리자 전용 라우팅: `/admin/dashboard`
