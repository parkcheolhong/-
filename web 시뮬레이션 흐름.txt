## 🌍 Web 프리뷰 데모 시뮬레이션 흐름

### 1️⃣ 홈 화면: 브랜드 소개 + 감정 카드 진입

- **화면 구성**:  
  - 상단 로고 + 슬로건 (“표현으로 연결되는 세계”)  
  - 감정 카드 6종 (기쁨, 슬픔, 분노, 놀람, 평온, 혼란)  
  - “표현 시작하기” 버튼

- **사용자 행동**:  
  - 감정 카드 클릭 → 해당 감정의 표현 카드로 진입

- **기능 반응**:  
  - 카드 확대 + 표현 예시 출력  
  - “피드백 남기기” 버튼 활성화

---

### 2️⃣ 챌린지 화면: 미션 수행 + XP 획득

- **화면 구성**:  
  - 오늘의 표현 미션 (“슬픔을 표현해보세요”)  
  - 음성 녹음 버튼 + 텍스트 입력창  
  - XP 게이지 + 공유 버튼

- **사용자 행동**:  
  - 음성 녹음 또는 텍스트 입력  
  - 미션 완료 후 공유

- **기능 반응**:  
  - XP 상승 애니메이션  
  - 피드에 자동 업로드

---

### 3️⃣ 지도 화면: 글로벌 표현 흐름 시각화

- **화면 구성**:  
  - 세계 지도 + 국가별 표현 카드  
  - 국가 클릭 시 해당 표현 예시 출력

- **사용자 행동**:  
  - 국가 선택 → 표현 흐름 확인  
  - “내 표현 추가” 버튼 클릭

- **기능 반응**:  
  - 사용자 표현이 지도에 추가됨  
  - 실시간 업데이트 반영

---

### 4️⃣ 인터뷰 화면: XR 기반 감정 대화 체험

- **화면 구성**:  
  - 인터뷰 질문 카드 (“당신은 언제 가장 평온했나요?”)  
  - XR 인터뷰 영상 + 음성 인식 버튼

- **사용자 행동**:  
  - 질문에 답변 녹음 또는 텍스트 입력  
  - XR 영상 감상

- **기능 반응**:  
  - GPT 기반 소개글 자동 생성  
  - 인터뷰 피드에 업로드

---

### 5️⃣ 피드 화면: 표현 공유 + 댓글 + 좋아요

- **화면 구성**:  
  - 사용자 표현 카드 리스트  
  - 좋아요 버튼 + 댓글 입력창

- **사용자 행동**:  
  - 표현 카드 클릭 → 상세 보기  
  - 댓글 작성 또는 좋아요 클릭

- **기능 반응**:  
  - 실시간 반응 업데이트  
  - 표현력 성장 리포트에 반영
