## 🛠️ Web 프리뷰 데모 실제 구현 설계안

### 1️⃣ 프로젝트 구조

```bash
nadotong-preview/
├── index.html
├── style.css
├── script.js
├── assets/
│   ├── images/
│   └── icons/
├── components/
│   ├── emotionCards.html
│   ├── challenge.html
│   ├── map.html
│   ├── interview.html
│   └── feed.html
```

- `index.html`: 전체 페이지 라우팅  
- `style.css`: 공통 스타일  
- `script.js`: 사용자 인터랙션 처리  
- `components/`: 각 기능별 화면 분리  
- `assets/`: 이미지, 아이콘, 배경 요소

---

### 2️⃣ 주요 기능 흐름

| 기능 | 설명 |
|------|------|
| 감정 카드 선택 | 카드 클릭 → 표현 예시 출력 + 피드백 버튼 |
| 챌린지 수행 | 미션 카드 → 음성 녹음/텍스트 입력 → XP 획득 |
| 글로벌 지도 | 국가 클릭 → 표현 흐름 시각화 + 사용자 표현 추가 |
| 인터뷰 체험 | 질문 카드 → XR 영상 감상 + 답변 입력 |
| 피드 공유 | 표현 카드 리스트 → 좋아요/댓글 기능 |

---

### 3️⃣ 구현 방식 제안

- **HTML**: 각 섹션을 `section` 태그로 분리  
- **CSS**: 감정별 색상 테마 적용 (예: 슬픔=블루, 기쁨=옐로우)  
- **JavaScript**:  
  - 카드 클릭 이벤트  
  - XP 게이지 애니메이션  
  - 지도 인터랙션 (Leaflet.js 또는 D3.js 활용 가능)  
  - 인터뷰 영상 재생 + 텍스트 입력 처리  
  - 피드 댓글/좋아요 기능

---

### 4️⃣ 배포 방식
