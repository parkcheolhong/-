## ✅ 구성된 사용자 테스트 시나리오 요약

| 테스트 항목 | 사용자 행동 흐름 | 시스템 반응 |
|-------------|------------------|--------------|
| 표현 카드 | 감정 선택 → 표현 입력 → 피드백 확인 | 예시 출력 + 피드백 버튼 |
| 챌린지 | 미션 수행 → XP 획득 → 공유 | XP 애니메이션 + 피드 업로드 |
| 지도 | 국가 선택 → 표현 흐름 확인 → 표현 추가 | 지도 시각화 + 실시간 반영 |
| 인터뷰 | 질문 응답 → XR 영상 감상 → 소개글 생성 | GPT 소개글 자동 삽입 |
| 피드 | 카드 탐색 → 좋아요/댓글 | 리포트 반영 + 실시간 반응 |
