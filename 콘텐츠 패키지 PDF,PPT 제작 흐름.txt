## 🧩 콘텐츠 패키지 PDF/PPT 제작 흐름

### 1️⃣ 구성 항목별 슬라이드 분류

| 항목 | 슬라이드 수 | 내용 요약 |
|------|--------------|-----------|
| 브랜드 개요 | 2 | 슬로건, 철학, 문제의식 |
| 콘텐츠 맵 | 3 | 표현 카드 → 챌린지 → 지도 → 인터뷰 → 피드 |
| 실행 흐름 | 3 | 디자인 → 배포 → 테스트 → 개선 |
| Web 프리뷰 | 2 | 데모 화면 구성 + 기능 흐름 |
| 사용자 테스트 | 2 | 시나리오 + 피드백 수집 방식 |
| 개선 제안서 | 2 | 기능별 개선 방향 + 실행 우선순위 |
| 협업 구조 | 2 | GitHub / Notion / 일정표 / 역할 분배 |

총 약 **16~18 슬라이드** 예상

---

### 2️⃣ 디자인 요소

- 브랜드 컬러 기반 배경  
- 감정별 아이콘 삽입 (기쁨, 슬픔, 분노 등)  
- Web 프리뷰 화면 캡처 삽입  
- 사용자 피드백 인용 포함  
- 슬라이드별 애니메이션 최소화 (가독성 중심)

---

### 3️⃣ 제작 방식

- **PDF 버전**: 제안서, 콘텐츠 맵, 개선안 보고서용  
- **PPT 버전**: IR 발표, 교육기관 미팅, 캠페인 설명용  
- **제작 툴**: PowerPoint, Keynote, Figma, Canva 등 가능  
- **배포 방식**:  
  - GitHub 저장소 업로드  
  - Notion 전략 페이지 삽입  
  - 이메일/링크 공유

---

### 4️⃣ 활용 대상

- 교육기관  
- 콘텐츠 기업  
- 창작자 커뮤니티  
- SNS 캠페인 파트너  
- IR 발표 및 제안 미팅
