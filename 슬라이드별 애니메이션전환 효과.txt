## 🎞️ 슬라이드별 애니메이션·전환 효과 제안

| 슬라이드 제목 | 애니메이션 효과 | 전환 방식 |
|----------------|------------------|------------|
| 브랜드 개요 | 감정 카드가 하나씩 떠오르는 페이드 인 | 페이드 (Fade) |
| 문제의식 | 말풍선이 끊어졌다가 연결되는 선 드로잉 | 와이프 (Wipe) |
| 콘텐츠 맵 | 흐름도 요소가 순차적으로 나타나는 줌 인 | 푸시 (Push) |
| 표현 카드 | 카드가 하나씩 회전하며 등장 | 플립 (Flip) |
| 챌린지 | XP 게이지가 채워지는 애니메이션 | 슬라이드 업 (Slide Up) |
| 지도 | 국가별 표현 아이콘이 점점 나타나는 스케일 인 | 줌 (Zoom) |
| 인터뷰 | 질문 카드가 하나씩 펼쳐지는 페이드 인 | 커버 (Cover) |
| 피드 | 카드형 피드가 아래에서 올라오는 슬라이드 인 | 슬라이드 (Slide) |
| Web 프리뷰 데모 | 데모 화면이 점점 확대되며 보여짐 | 줌 인 + 페이드 |
| 사용자 테스트 | 체크리스트가 체크되며 나타나는 팝업 | 페이드 + 팝 |
| 기능 개선 제안 | 개선 전/후 비교표가 좌우로 나뉘며 등장 | 슬라이드 좌우 (Split) |
| 협업 구조 | 역할 아이콘이 하나씩 나타나는 드롭 인 | 드롭 (Drop) |
